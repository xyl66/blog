# 预编译阶段

js代码是运行时编译（编译发生在代码执行前几微秒），即预编译完立即执行。预编译阶段进行初始化执行上下文

  • 声明提升

    • 初始化上下文时有变量声明，先去作用域链查找是否已存在，无则创建属性并赋值undefined，有则不作处理
    • 初始化上下文时有函数声明，先去作用域链查找是否已存在，无则创建属性并赋值函数，有则覆盖
  function a(b) {
    console.log(b) 
    function b() {
      console.log(b)
    }
    b()
  }
  a(2)
