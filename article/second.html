# css选择器权重

# css权重分为5级

  1. 第一级：内联样式<div style=""></div> 权重 ==1000==
  2. 第二级：id选择器 #开头 权重==0100==
  3. 第三级：class选择器 权重==0010==
  4. 第四级：标签选择器、伪类权重==0001==
  5. 第五级：通用选择器(*),子选择器(>)和相邻同胞选择器(+)无权重==0000==

# 示例

<p class="p1">这是内容</p>
<style>
	p.p1{ // 权重0011
	color:blue
	}
	.p1{ // 权重0010
	color:red
	}
</style>
